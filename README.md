## News Portal - JSON-Server Backend

A simple CRUD News Portal application using JSON-Server as a mock backend.

## Features

- User Authentication (Simulated with dropdown selection)
- Create News - Add new news articles
- View News - Browse all news with author information
- News Details - Read full article with comments
- Add Comments - Comment on news articles
- Edit/Delete News - Authors can modify their own articles
- Search Functionality - Filter news by title
- Responsive Design - Works on all devices

## Technologies Used

- Frontend: HTML, CSS,JavaScript
- Backend: JSON-Server (Mock REST API)
- Data Storage: JSON file-based database

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/morshaline1504/NEWS-PORTAL.git

2. Navigate to project folder:
   cd NEWS-PORTAL

3. Install dependencies:

   npm install
   
   npm install json-server@0.17.4
   
   npm install --save-dev concurrently open-cli

4. Start JSON Server:
   npm start
